# Tutorial Interactivo de Electrónica

Sitio web educativo interactivo para aprender electrónica desde los fundamentos hasta la práctica con simulación.

## 🎯 Características

- **15 Temas Completos**: Desde los orígenes de la electrónica hasta el taller de simulación práctica
- **Calculadoras Interactivas**: Ley de Ohm, potencia, frecuencia, resistencias serie/paralelo, código de colores
- **Visor 3D de Componentes**: Visualiza componentes electrónicos en 3D con rotación interactiva
- **Simulador Tinkercad Integrado**: 8 ejercicios prácticos con instrucciones paso a paso
- **Diagramas Esquemáticos SVG**: Diagramas técnicos limpios y profesionales
- **Diseño Responsive**: Funciona perfectamente en desktop, tablet y móvil
- **Progreso del Curso**: Sistema de seguimiento de progreso automático

## 📚 Contenido del Curso

### Temas Teóricos (1-14)
1. Orígenes de la Electrónica
2. Evolución de la Electrónica
3. Conceptos Básicos de Electricidad
4. Componentes Pasivos
5. Componentes Activos
6. Circuitos Integrados
7. Herramientas y Equipos (7.1-7.5)
8. Leyes Fundamentales
9. Análisis de Circuitos (9.1-9.5)
10. Electrónica Digital (10.1-10.7)
11. Electrónica Analógica (11.1-11.7)
12. Fuentes de Alimentación (12.1-12.5)
13. Aplicaciones Prácticas (13.1-13.3)
14. Simulación y Prototipado (14.1-14.2)

### Taller de Simulación (Tema 15)
15.1. **Mediciones Eléctricas**: Uso del multímetro  
15.2. **Resistencias**: Circuitos serie, paralelo y mixto  
15.3. **Condensadores**: Carga y descarga de capacitores  
15.4. **Transistores**: Transistor como interruptor  
15.5. **Circuitos Integrados**: Oscilador con CI 555  
15.6. **Juego de Luces con Transistor**: Secuencia de LEDs  
15.7. **Juego de Luces con 555**: Oscilador astable  
15.8. **Contador 0-9**: 555 + 7490 + 7447 + Display 7 segmentos

## 🚀 Cómo Usar

1. **Abrir el sitio**: Simplemente abre `index.html` en tu navegador web moderno
2. **Navegar**: Usa el menú lateral para navegar entre temas
3. **Calculadoras**: Ingresa valores en las calculadoras para obtener resultados instantáneos
4. **Visor 3D**: Haz clic en componentes para verlos en 3D con controles de rotación
5. **Simulador**: En el Tema 15, haz clic en cada ejercicio para abrir el simulador Tinkercad

## 🎓 Requisitos

- Navegador web moderno (Chrome, Firefox, Edge, Safari)
- Conexión a internet (para Tinkercad)
- Cuenta gratuita de Tinkercad (para realizar ejercicios prácticos)

## 🔧 Funcionalidades Principales

### Calculadoras Interactivas
- **Ley de Ohm**: Calcula resistencia a partir de voltaje y corriente
- **Potencia**: Calcula potencia eléctrica
- **Frecuencia**: Convierte período a frecuencia
- **Resistencias Serie**: Calcula resistencia total en serie
- **Resistencias Paralelo**: Calcula resistencia total en paralelo
- **Código de Colores**: Decodifica resistencias por bandas de color

### Visor 3D
- Rotación en ejes X, Y, Z
- Zoom in/out
- Reset de vista
- Especificaciones técnicas de cada componente

### Integración Tinkercad
- Modal con instrucciones detalladas
- Lista de materiales interactiva
- Diagramas esquemáticos
- Instrucciones paso a paso
- Checklist de verificación
- Acceso a proyecto de referencia completado
